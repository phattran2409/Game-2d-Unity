# 🌞 SunnyLand 2D Game

Welcome to **SunnyLand** — a vibrant 2D platformer game built with Unity! Explore pixel art landscapes, overcome obstacles, and enjoy classic platforming fun.

---

## 🎮 Game Features
- Beautiful pixel art graphics
- Smooth character movement
- Multiple levels to explore
- Collectibles and power-ups
- Engaging sound effects and music

---

## 🚀 Getting Started
1. **Clone or Download** this repository.
2. Open the project in [Unity](https://unity.com/) (recommended version: 2021.3 LTS or newer).
3. Open the `SunnyLand` scene from the `Assets/Scenes` folder.
4. Press `Play` to start the adventure!

---

## 🕹️ Controls
- **Arrow Keys / WASD**: Move
- **Space**: Jump
- **Esc**: Pause
- **Attack**  : Click mouse
---

## 🛠️ Project Structure
- `Assets/` — Game assets, scripts, scenes, prefabs, and artwork
- `ProjectSettings/` — Unity project settings
- `Packages/` — Package dependencies

---

## 📦 Dependencies
- Unity Input System
- Universal Render Pipeline (URP)
- TextMesh Pro
